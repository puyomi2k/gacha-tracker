v1.0 | 25. Mai 2025

[NOTE] Initial Release of the Gacha-Tracker
[ADDED] Support for Genshin und Honkai: Start Rail through StarDB aswell as for Wuthering Waves via WuWa Tracker
[ADDED] Workaround for shutting up the annoying BIOS-Beep-Sound (Thats why the App beg for Admin permissions)


v2.0 | 28. Juni 2025

[ADDED] Multiple Options for Genshin and HSR inclusive Submenus
[ADDED] ZZZ Support - currently only for StarDB
[ADDED] Menu Closing automatically - App Closes after 60 Seconds without input
[CHANGED] Minor changes on Menu design and Select options
[REMOVED] Admin permissions not longer required. This was previously required to stop the BEEP Service permanently, to shut up the annoying BIOS Beep Sounds when Pressing an invalid key - Windows does replace it with the regular sounds when disabled (Still can be done manually by typing "net stop beep" into cmd - with Admin permissions ofc)


v25.9.1 (v2.1) | 01. September 2025

[ADDED] introduced Changelog (partially for older Versions aswell)
[CHANGED] Switching and changing apperance of Version Display and dev credit in the menu header
[CHANGED] Version format changed to Year-Month-Day (YY.M.D)
[CHANGED] Links to stardb.gg modified (opening the Page now directly opens the pull import page)
[REMOVED] Auto closing of the App after 60 Seconds idling removed. The Submenu timeout isn't affected by this
[NOTE] In future updates language support will be added (English will be first - and prolly the only one for a while). The first step towards we did by providing an english version of the changelog
